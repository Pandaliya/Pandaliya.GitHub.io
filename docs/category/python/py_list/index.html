    python数组（list） #

    list是python内置的一种数据类型，他是一种顺序集合，可以添加和删除元素，其中元素类型不必相同。内部保存的是数据的指针。

    list的增加元素的方式 #

    append

    append用来向list末尾追加单个元素（参数为集合时，也视为单个元素）。

    insert

    insert用来将单个元素插入到list中，参数包括插入点索引和插入元素

    extend

    extend用来链接list，将参数集合的元素降维添加到list中
